Privacy Policy

Effective Date: May 12, 2026

This Privacy Policy describes how the Security Deposit Recovery browser extension handles information.

1. Data Collection

The extension does not collect, sell, transmit, or share personal information with any project-owned server. There is no account system, backend service, analytics service, advertising service, or AI/LLM API integration.

2. Local Data Usage and Storage

The extension stores case information locally in the browser's isolated extension storage using chrome.storage.local. This can include state selection, deposit and rent amounts, move-out dates, landlord action details, evidence checklist values, contact details, and refund instructions entered by the user.

This local data is used only to preserve form progress, evaluate the case with local deterministic rules, and generate a demand letter PDF on the user's device.

3. Downloads and Notifications

The extension uses the browser downloads API to save a generated PDF demand letter when the user requests it. It uses browser notifications only to confirm document generation or remind the user to follow up.

4. Third-Party Services

The extension does not make third-party network requests and does not load remote code. All rules, validation, storage, and PDF generation run locally inside the extension.

5. Data Sharing

We do not sell, transfer, or share user data. Information entered into the extension remains in local browser extension storage unless the user clears it or removes the extension.

6. Legal Notice

The extension provides document preparation assistance and informational guidance only. It is not legal advice and does not create an attorney-client relationship.

7. Changes to This Policy

We may update this Privacy Policy by posting a new version on this page.
